Introdução ao CSS

Objetivo de Aprendizagem

Ao finalizar este recurso educacional, o estudante será capaz de compreender o conceito de CSS e identificar sua importância na construção de páginas web.

O que é CSS?

CSS significa Cascading Style Sheets, ou Folhas de Estilo em Cascata.

O CSS é uma linguagem utilizada para definir a aparência dos elementos HTML de uma página web. Com CSS podemos modificar cores, fontes, tamanhos, espaçamentos e posicionamento dos elementos.

CSS e HTML trabalham juntos

O HTML cria a estrutura da página, enquanto o CSS define sua apresentação visual.

Exemplo: 


HTML:
<h1>Olá Mundo</h1>


CSS:

h1 {
    color: blue;
}
