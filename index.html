Introdução ao CSS

Objetivo de Aprendizagem

Ao finalizar este recurso, o estudante será capaz de compreender o conceito de CSS, identificar sua função no desenvolvimento web e aplicar estilos básicos em páginas HTML.

1. O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). É uma linguagem utilizada para definir a aparência dos elementos HTML.

Com CSS podemos modificar:

  • Cores
  • Fontes
  • Tamanhos
  • Espaçamentos
  • Posicionamento dos elementos

2. HTML e CSS trabalham juntos

O HTML cria a estrutura da página. O CSS define sua aparência visual.

Este texto foi estilizado usando CSS.

3. Seletores CSS

Seletores indicam quais elementos HTML receberão determinado estilo. 


Exemplo:

p {

color: blue;

}

4. Classes e IDs

As classes permitem aplicar estilos em vários elementos. Os IDs identificam um elemento específico. 


.classe {

color:red;

}


#identificador {

font-size:20px;

}
